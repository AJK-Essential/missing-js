## 💰 Licensing & Commercial Use

Missing JS uses a **Sustainable Open Source** model.

### 1. Personal & Educational (Always Free)

If you are a student, a hobbyist, or working on a non-commercial project, **all components are free to use.**

### 2. Commercial Use

For-profit entities using the **Virtualizer** series require a one-time Commercial License.
