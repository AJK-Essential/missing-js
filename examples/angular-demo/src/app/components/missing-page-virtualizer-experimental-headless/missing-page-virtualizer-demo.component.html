Right now only swipe scroll has been implemented in this version. This is a demo of a paged-virtualizer component that can technically remember the heights of 10^6 pages. If u inspect the DOM, only a few DOM elements are there that simulate the whole feed. This component is frontend-framework agnostic as it is a web-component and the DOM that u see (the cards are fully provided by the consuming framework) and so, the consuming framework has full access to the DOM.
