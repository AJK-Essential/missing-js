This is a demo of a paged-virtualizer component that can technically remember the heights of 10^6 pages. If u inspect the DOM, only a few DOM elements are there that simulate the whole feed. This component is frontend-framework agnostic as it is a web-component and the DOM that u see (the cards are fully provided by the consuming framework) and so, the consuming framework has full access to the DOM.
The cards with the content have been generated automatically using a script by combining several phrases together to simulate somewhat realistic social-media posts.
Right now we are only showing 100,000 pages, each with 10 items, and thereby virtualizing 10 ^ 5 * 10 = 10 ^ 6 DOM Nodes. Scroll all the way below to see the load more button to load more till u reach the 1000000 (1M) post.
